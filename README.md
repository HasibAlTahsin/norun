## What is NORUN?

**NORUN** lets you run Windows applications on Linux easily using **Wine / Proton** with:

- CLI + optional GUI
- Virtual desktop support
- DXVK toggle
- Isolated Wine prefixes
- Desktop shortcut generation
- AppImage support (portable, no install)
